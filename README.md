# Proposta Personalizada — Dr. Luiz Sarmanho

## Objetivo

Montar uma proposta comercial no formato inspirado no case Hanna/Ornato, mas aplicada ao contexto do Dr. Luiz e usando a identidade visual aprovada da marca.

## Entregavel principal

- `index.html` — proposta visual/comercial em pagina unica

## Estrutura da proposta

1. Hero comercial com contexto e urgencia
2. Diagnostico resumido com 3 gargalos principais
3. Preview do site de captacao
4. Aplicacao da identidade visual
5. Preview de posts
6. Lista completa de entregas Pulso
7. Investimento com recomendacao de pacote
